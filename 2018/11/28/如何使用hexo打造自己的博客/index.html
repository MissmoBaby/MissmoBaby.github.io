准备环境

  1. Node.js下载，并安装，详细步骤：https://www.simon96.online/2018/11/10/hexo-env/

  2. Git下载，并安装，详细步骤：https://www.simon96.online/2018/11/10/hexo-env/

  3. 安装Hexo，并在命令行（即Git Brash）运行以下命令：

    npm install -g hexo-cli

    • hexo相关命令均在站点目录下，用Git Bash运行。

      路径为<folder>\config.yml

    • 主题配置文件：站点目录下的themes文件夹下，主题文件夹下的_config.yml。

      路径为<folder>\themes\<主题文件夹>\_config.yml

    1. 启动服务器。在路径下，命令行（即Git Brash）输入以下命令，运行即可：

      hexo server

    2. 浏览器访问网址：http://localhost:4000/

    至此，您的Hexo博客已经搭建在本地。
